# Morning Note

生成简洁的每日晨会笔记，覆盖隔夜市场动态、关键事件和交易想法。

## Data Source Priority

### Layer 1: MCP
- **coingecko** — 加密市场隔夜表现、重大价格变动

### Layer 2: Chrome CDP
- `fred.stlouisfed.org/series/{series_id}` — 经济数据发布（利率变动、就业数据等）
- `defillama.com/protocol/{protocol}` — DeFi TVL 变动、协议重大事件

### Layer 3: Web Search
- 财经新闻（earnings, M&A, 政策变化）
- 期货/盘前数据
- 加密新闻（项目公告、监管动态）

## Workflow

### Step 1: Overnight Developments

扫描隔夜/盘前重大事件：

**传统市场:**
- 盈利发布（beat/miss + 关键指标）
- 指引变更（上调/下调/维持）
- M&A 公告或传闻
- 分析师评级变化
- 宏观数据或政策变化

**加密市场:**
- BTC/ETH 隔夜价格变动（>3% 标注）
- 重大项目公告（代币发行、空投、升级）
- DeFi 协议重大事件（黑客、TVL 大幅变动）
- 监管动态（SEC、各国政策）
- 稳定币脱锚或大额铸造/赎回

**宏观/市场环境:**
- 隔夜期货/盘前表现
- 美元/商品/国债收益率变动
- 当日经济数据发布预告

### Step 2: Compile Morning Note

格式要求：**2 分钟内可读完**。

```
**[YYYY-MM-DD] 晨会笔记**

**头条: [最重要的一件事]**
- 2-3 句说明事件及影响
- 对持仓/关注标的的影响

**隔夜/盘前动态**
- [标的A]: 一句话总结 + 观点
- [标的B]: 一句话总结 + 观点
- [宏观/行业]: 板块级别动态

**加密市场**
- BTC/ETH: 价格变动 + 原因
- [项目/协议]: 重大事件
- 监管/政策: 如有

**今日关注**
- [时间]: 事件 + 预期 vs 我们的判断
- [时间]: 经济数据发布
- [时间]: 项目里程碑

**交易想法**（如有）
- [做多/做空/观望] [标的]: 1-2 句逻辑 + 催化剂
- 风险: 什么情况会证伪
```

## Output Format

- **Primary**: Markdown（对话中直接显示，或保存为 `Morning_Note_{YYYYMMDD}.md`）
- 保持 1 页以内
- 有观点 — 只总结不给观点的晨报没有价值

## Quality Checklist

- [ ] 最重要的事件放在头条位置
- [ ] 传统市场和加密市场都已覆盖
- [ ] 事件有观点（不只是转述新闻）
- [ ] 今日关注包含具体时间
- [ ] 交易想法包含风险提示
- [ ] 数据来自实时源，非训练数据
- [ ] 如无重大事件，简短说明"隔夜无重大变化"即可

## Important Notes

- 有观点 — 只转述新闻的晨报没有价值
- 头条优先 — 最重要的事情放在最前面
- "无重大事件"也是有效的晨报 — 说"隔夜无重大变化，维持现有布局"
- 区分信号和噪音 — 不是所有新闻都值得提及
- 标注时效 — 如果是盘前写的，注明开盘后可能变化
- 如果之前的判断错了，在下一期晨报中坦然承认
